# machineuse.dotcommoners.com

Landing + marketing site for **[Machineuse](https://github.com/dotcommoners/machineuse)** — distributed container management for browser automation at scale, by [dotcommoners](https://github.com/dotcommoners).

Built with [Astro](https://astro.build) as a static, SEO/AI-search-optimized site. Served under the docs domain at **/machineuse/**.

## Develop

```bash
npm install
npm run dev
npm run build    # static output to ./dist (base path: /machineuse/)
```
